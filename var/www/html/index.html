About the Project

This web server is built as a learning project to understand HTTP protocols, network programming, and C++ development. It implements core features of HTTP/1.1 while maintaining high performance and stability.

Key Features

  • Non-blocking I/O operations
  • Configurable virtual hosts
  • Custom error pages
  • Directory listing
  • File upload handling
